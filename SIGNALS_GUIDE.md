# 📊 PolySniper 信号与颜色说明

## 🎨 比赛卡片颜色含义

比赛卡片的边框和背景颜色**仅表示主队获胜概率**（基于 Polymarket 价格）：

### 🟢 绿色卡片（Bullish）
- **条件**：主队价格 ≥ 60¢
- **含义**：市场认为主队大概率赢
- **显示**：绿色边框 + 绿色背景 + "高概率"标签

### 🔴 红色卡片（Bearish）
- **条件**：主队价格 ≤ 40¢  
- **含义**：市场认为主队大概率输
- **显示**：红色边框 + 红色背景 + "低概率"标签

### ⚪ 灰色卡片（Neutral）
- **条件**：主队价格在 40-60¢ 之间
- **含义**：双方势均力敌
- **显示**：灰色边框 + "等待中"标签

> **注意**：卡片颜色≠交易信号！它只是价格高低的视觉提示。

---

## 💡 交易策略信号

策略信号基于**价格 + 比分 + 时间**三重因素分析，才是真正的交易建议：

### 🟢 强买入 (STRONG_BUY) 🔔🔔
**触发条件（所有条件同时满足）：**
1. **价格区间**：35¢ - 45¢
2. **比分状态**：落后 1-6 分
3. **时间窗口**：第 1-3 节（或第4节前5分钟）
4. **置信度**：≥ 70%

**含义**：黄金进场点！价格被低估，有反弹空间。

**提示音**：🔔🔔 双音（叮叮）

---

### 🟢 买入 (BUY)
**触发条件：**
- 价格 35¢ - 45¢ + 落后 1-6 分（但时间或其他条件不完美）
- 置信度：50-70%

**含义**：可以考虑，但不如强买入稳。

**提示音**：无

---

### 🔴 强卖出 (STRONG_SELL) 🔔
**触发条件：**
1. **价格区间**：≥ 75¢
2. **比分状态**：大幅领先
3. **置信度**：≥ 70%

**含义**：价格过高，建议止盈。

**提示音**：🔔 单音（叮）

---

### 🔴 卖出 (SELL)
**触发条件：**
- 价格 ≥ 65¢ + 领先（但不够强）
- 置信度：50-70%

**含义**：可以考虑止盈。

**提示音**：无

---

### ⚪ 中性 (NEUTRAL)
**含义**：不符合任何交易条件，观望。

**提示音**：无

---

## 🔔 提示音说明

### 何时播放？
只有**新的强信号**（STRONG_BUY 或 STRONG_SELL）出现时才播放提示音。

### 音效区别
- **🟢 强买入**：双音（叮~叮~）高频 800Hz
- **🔴 强卖出**：单音（叮~）低频 600Hz

### 不会重复播放
同一场比赛、同一球队、同一类型的信号不会重复播放提示音。

## 📈 核心策略逻辑

### 均值回归 (Mean Reversion)
当一支球队：
- **暂时落后** 1-6 分（小分差）
- **价格被低估** 35-45¢
- **还有时间** 前3节

→ 价格有很大概率反弹回归正常水平（止盈点 75¢+）

### 波段交易 (Swing Trading)
在比赛中盘捕捉价格波动：
- **低吸**：落后时买入（35-45¢）
- **高抛**：领先时卖出（75¢+）

## 🎲 ESPN 胜率预测

PolySniper 整合了 ESPN 的胜率预测数据，帮助你判断市场价格是否合理：

### 赛前预测（基于赔率）
比赛开始前，系统会从 ESPN BET 赔率计算隐含胜率：

**计算公式**：
- **负数赔率**（热门队）：胜率 = |赔率| / (|赔率| + 100)
  - 例如：-150 → 60% 胜率
- **正数赔率**（冷门队）：胜率 = 100 / (赔率 + 100)
  - 例如：+200 → 33.3% 胜率

**应用场景**：
```
ESPN 赛前预测：主队 65% 胜率
Polymarket 价格：主队 45¢

分析：市场价格明显低于 ESPN 预测
信号：可能是买入机会 🟢
```

### 赛中实时胜率（Live Win Probability）
比赛开始后，ESPN 会根据实时比赛进展计算动态胜率：

**更新频率**：每次重要比赛事件后更新（得分、犯规、暂停等）

**应用场景**：
```
比赛进行中：
- 主队落后 3 分
- ESPN 实时胜率：52%（仍看好主队）
- Polymarket 价格：38¢（市场恐慌）

分析：价格与实际胜率偏离
信号：强买入机会 🟢🔔🔔
```

### 胜率 vs 价格偏差分析

系统会自动计算 ESPN 胜率与市场价格的偏差：

| 偏差程度 | 说明 | 交易信号 |
|---------|------|---------|
| > +15% | ESPN 胜率远高于市场价格 | 🟢 买入机会 |
| +5% ~ +15% | 轻微低估 | 🟢 谨慎买入 |
| -5% ~ +5% | 价格合理 | ⚪ 观望 |
| -15% ~ -5% | 轻微高估 | 🔴 谨慎卖出 |
| < -15% | 市场价格远高于 ESPN 胜率 | 🔴 卖出机会 |

**示例**：
```
ESPN 胜率：70%
市场价格：52¢ (52%)
偏差：+18%

结论：市场低估了主队获胜概率，
      可能是买入机会 🟢
```

### 注意事项

⚠️ **ESPN 胜率不是绝对真理**：
- 模型基于历史数据和统计概率
- 无法预测突发事件（伤病、裁判误判）
- 应结合比分、价格、时间综合判断

⚠️ **赛前 vs 赛中**：
- 赛前预测基于博彩赔率（市场共识）
- 赛中实时胜率更准确（考虑实际比赛进展）

## 💰 如何使用

### 1. 观察比赛卡片
- 绿色/红色只是价格指示，不是买卖信号

### 2. 等待强信号 🔔
- 听到提示音 → 查看策略信号卡片
- 确认信号详情：置信度、目标价位、止损

### 3. 查看价格趋势图
- 点击比赛卡片 → 查看该场比赛的价格走势
- 在主队/客队之间切换

### 4. 查看信号历史
- 在信号日志中查看所有历史信号
- 分析成功率和模式

---

## ⚠️ 风险提示

1. **策略信号仅供参考**，不构成投资建议
2. **价格可能不会按预期变化**，需要设置止损
3. **强信号不等于必赢**，只是概率较高
4. **建议小额测试**，验证策略有效性后再增加投入
5. **NBA比赛瞬息万变**，需要实时监控
