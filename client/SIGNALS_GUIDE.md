# 📊 PolySniper 信号与颜色说明

## 🎨 比赛卡片颜色含义

比赛卡片的边框和背景颜色**仅表示主队获胜概率**（基于 Polymarket 价格）：

### 🟢 绿色卡片（Bullish）
- **条件**：主队价格 ≥ 60¢
- **含义**：市场认为主队大概率赢
- **显示**：绿色边框 + 绿色背景 + "高概率"标签

### 🔴 红色卡片（Bearish）
- **条件**：主队价格 ≤ 40¢  
- **含义**：市场认为主队大概率输
- **显示**：红色边框 + 红色背景 + "低概率"标签

### ⚪ 灰色卡片（Neutral）
- **条件**：主队价格在 40-60¢ 之间
- **含义**：双方势均力敌
- **显示**：灰色边框 + "等待中"标签

> **注意**：卡片颜色≠交易信号！它只是价格高低的视觉提示。

## 交易策略信号

策略信号基于**价格 + 比分 + 时间 + ESPN胜率**四重因素分析，才是真正的交易建议：

### 强买入 (STRONG_BUY) - 强队抄底
**核心策略**：识别**赛前强队**的暂时价格低估机会

## 重要提示

### 不会生成信号的情况

1. **垃圾时间** 🗑️
   - 最后5分钟 + 分差 > 15分 → 比赛已定，无信号

2. **大分差** 📊
   - 任何时段分差 > 20分 → 比赛失控，无信号

3. **第4节大落后** ⏱️
   - 第4节落后 > 6分 → 逆转概率极低，不生成买入信号

4. **胜率不支持** ⚠️ 🆕
   - **强买入**：要求赛前胜率 ≥ 50% 或 实时胜率 ≥ 45%
   - **买入**：要求赛前胜率 ≥ 45% 或 实时胜率 ≥ 40%
   - **如果胜率太低，不会推荐买入！**

### 强买入 (STRONG_BUY) - 强队抄底
**触发条件：**
1. **赛前判断**：ESPN赛前胜率 ≥ 55%（确认为强队）
2. **价格低估**：当前价格 < 赛前胜率 - 10%（例如：赛前70%预期，现价 < 60¢）
3. **比分可接受**：
   - 暂时落后但分差不大（< 10分）
   - 或者比分胶着/领先
4. **时间充裕**：比赛早期（前3节），逆转机会大
5. **置信度**：≥ 70%

**策略逻辑**：
- 强队的赛前胜率代表真实实力
- 暂时落后导致市场恐慌性抛售，价格被低估
- 强队有很大概率追回比分，价格回归预期

**示例**：
```
赛前ESPN胜率：湖人 68%
当前比分：湖人落后 5分（第2节）
Polymarket价格：湖人 52¢

分析：强队暂时落后，价格低于赛前预期
信号：🟢 强买入 - 抄底机会！
```

**提示音**：已禁用（静音模式）

---

### 🟢 买入 (BUY)
**触发条件：**
- 价格 35¢ - 45¢ + 小幅落后（1-6分）
- 时间窗口：前3节
- 但不满足强队抄底条件（赛前胜率不够高或价格未明显低估）
- **胜率门槛**：赛前胜率 ≥ 45% 或 实时胜率 ≥ 40%
- 置信度：50-70%

**含义**：普通买入机会，风险相对较高。

**提示音**：无

---

### 🔴 强卖出 (STRONG_SELL) - 领先套现
**触发条件：**
1. **价格高位**：≥ 75¢
2. **比分优势**：大幅领先（≥ 8分）
3. **时间因素**：任何时段
4. **置信度**：≥ 70%

**含义**：价格已达高位，建议止盈套现。

**提示音**：已禁用（静音模式）

---

### 🔴 卖出 (SELL)
**触发条件：**
- 价格 ≥ 65¢ + 领先（但不够强）
- 置信度：50-70%

**含义**：可以考虑止盈。

**提示音**：无

---

### ⚪ 中性 (NEUTRAL)
**含义**：不符合任何交易条件，观望。

**提示音**：无

## 📈 核心策略逻辑

### 1. 强队抄底策略 (Strong Team Dip Buying)
**核心思想**：利用赛前胜率判断真实实力，在强队暂时落后时抄底

**优势**：
- 有明确的实力基准（赛前ESPN胜率）
- 不会盲目抄底弱队
- 强队逆转概率更高

**风险控制**：
- 仅在比赛早期（前3节）执行
- 分差不能太大（< 10分）
- 价格必须明显低于赛前预期

### 2. 均值回归 (Mean Reversion)
价格偏离真实胜率时会回归：
- **价格低于赛前预期** → 上涨空间
- **价格高于实际胜率** → 下跌风险

### 3. 波段交易 (Swing Trading)
捕捉比赛中的价格波动：
- **低吸**：强队落后时买入
- **高抛**：大幅领先时卖出（≥ 75¢）

## 🎲 ESPN 胜率预测

PolySniper 整合了 ESPN 的胜率预测数据，帮助你判断市场价格是否合理：

### 赛前预测（基于赔率）- 强队判断基准 ⭐
比赛开始前，系统会从 ESPN BET 赔率计算隐含胜率，并**永久缓存**（localStorage持久化）。

**计算公式**：
- **负数赔率**（热门队）：胜率 = |赔率| / (|赔率| + 100)
  - 例如：-150 → 60% 胜率
- **正数赔率**（冷门队）：胜率 = 100 / (赔率 + 100)
  - 例如：+200 → 33.3% 胜率

**核心作用**：作为**强队判断基准**
- 赛前胜率 > 65% → 认定为强队
- 即使比赛中暂时落后，也会作为抄底参考
- 缓存在localStorage，刷新页面不丢失

**应用场景**：
```
ESPN 赛前预测：湖人 70% 胜率（强队）
比赛进行中：湖人落后 4分
Polymarket 价格：湖人 48¢

分析：强队价格远低于赛前预期（70% vs 48%）
信号：🟢 强买入 - 强队抄底机会！
```

### 赛中实时胜率（Live Win Probability）
比赛开始后，ESPN 会根据实时比赛进展计算动态胜率：

**更新频率**：每次重要比赛事件后更新（得分、犯规、暂停等）

**应用场景**：
```
比赛进行中：
- 主队落后 3 分
- ESPN 实时胜率：52%（仍看好主队）
- Polymarket 价格：38¢（市场恐慌）

分析：价格与实际胜率偏离
信号：强买入机会 🟢🔔🔔
```

### 胜率 vs 价格偏差分析

系统会自动计算 ESPN 胜率与市场价格的偏差：

| 偏差程度 | 说明 | 交易信号 |
|---------|------|---------|
| > +15% | ESPN 胜率远高于市场价格 | 🟢 买入机会 |
| +5% ~ +15% | 轻微低估 | 🟢 谨慎买入 |
| -5% ~ +5% | 价格合理 | ⚪ 观望 |
| -15% ~ -5% | 轻微高估 | 🔴 谨慎卖出 |
| < -15% | 市场价格远高于 ESPN 胜率 | 🔴 卖出机会 |

**示例**：
```
ESPN 胜率：70%
市场价格：52¢ (52%)
偏差：+18%

结论：市场低估了主队获胜概率，
      可能是买入机会 🟢
```

### 注意事项

⚠️ **ESPN 胜率不是绝对真理**：
- 模型基于历史数据和统计概率
- 无法预测突发事件（伤病、裁判误判）
- 应结合比分、价格、时间综合判断

⚠️ **赛前 vs 赛中**：
- 赛前预测基于博彩赔率（市场共识）
- 赛中实时胜率更准确（考虑实际比赛进展）

## 💰 如何使用

### 1. 观察比赛卡片
- 绿色/红色只是价格指示，不是买卖信号

### 2. 关注强信号
- 查看策略信号卡片（顶部显示最强信号）
- 确认信号详情：置信度、目标价位、止损
- 查看控制台日志：`🔔 强买入/强卖出信号`

### 3. 查看价格趋势图
- 点击比赛卡片 → 查看该场比赛的价格走势
- 在主队/客队之间切换

### 4. 查看信号历史
- 在信号日志中查看所有历史信号
- 分析成功率和模式

---

## ⚠️ 风险提示

1. **策略信号仅供参考**，不构成投资建议
2. **价格可能不会按预期变化**，需要设置止损
3. **强信号不等于必赢**，只是概率较高
4. **建议小额测试**，验证策略有效性后再增加投入
5. **NBA比赛瞬息万变**，需要实时监控
